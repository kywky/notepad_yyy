# Notepad+ 简明使用说明

适用于 Notepad+ `0.3.27` 及之后版本。

## 1. 开始使用

打开应用后会先进入首页：

- **新建**：创建空白文本并进入编辑器。
- **打开**：从手机中选择任意文本、代码或日志文件。
- **打开文件夹**：选择一个文件夹，在侧边栏中浏览文件。
- **打开最近**：继续编辑最近使用的文件。
- 最近记录右侧的删除按钮可删除单项，**全部清空**可清除所有记录。
- 右上角太阳/月亮按钮用于切换明暗主题。

Android 也支持从文件管理器选择“打开方式”，或通过其他应用的“分享”将文本发送到 Notepad+。

## 2. 编辑器界面

顶部常用按钮从左到右为。手机窄屏会自动排成两行，宽屏显示为一行：

| 按钮 | 功能 |
| --- | --- |
| 新建 | 新建一个标签 |
| 打开 | 打开文件；网页版可一次选择多个文件 |
| 保存 | 覆盖保存当前文件；没有原文件位置时会提示选择位置 |
| 另存为 | 使用新名称或新位置保存当前文件 |
| 撤销 / 重做 | 撤回或恢复编辑操作 |
| 打开文件夹 | 选择文件夹并显示其中的文件 |
| 侧边栏 | 显示或隐藏文件夹侧边栏 |
| 查找 | 打开查找和替换面板 |
| 自动换行 | 开关长行自动换行；大文件模式下不可用 |
| 全屏 | 进入全屏专注模式 |

标题栏左侧点击 **Notepad+** 可返回首页；中间显示当前文件名，文件名前出现 `*` 表示尚未保存。右上角菜单保留显示设置：

- 切换明暗主题
- 调整字体大小（11～32）

长按没有文字的工具按钮，可以临时显示按钮名称。

## 3. 编辑与标签

- 直接点击编辑区即可输入和修改文字。
- 双指缩放可调整字体大小。
- 开启自动换行时，长行会按屏幕宽度显示；关闭后可左右滑动。
- 常见代码文件会根据文件名自动启用语法高亮，普通文件保持纯文本显示。
- 点击标签可切换文件，点击标签右侧 `×` 可关闭。
- 橙色圆点表示该标签有未保存的修改。
- 双击标签可重命名。
- 长按标签可选择：**重命名、关闭标签、关闭其他标签、关闭全部标签**。
- 关闭包含未保存内容的标签时，应用会先要求确认。

网页版还支持把文件直接拖入窗口打开。

## 4. 文件夹与侧边栏

选择文件夹后，点击侧边栏中的文件才会读取内容。可通过右上角菜单隐藏或重新显示侧边栏。

- 一个文件夹最多列出前 500 个文件。
- 手机窄屏下，打开文件后侧边栏会自动收起。
- 如果最近文件被移动、删除或权限失效，需要重新通过“打开”选择它。

## 5. 保存和另存为

- **保存**：有原文件权限时直接覆盖；否则进入另存为流程。
- **另存为**：点击工具栏中保存旁边的按钮，可选择新名称和新位置。
- 保存成功后底部会显示“文件已保存”。
- 返回首页不会丢弃当前标签，但退出或关闭未保存文件前应先保存。

## 6. 查找和替换

点击放大镜按钮打开面板：

- 在“查找内容”中输入关键词，所有匹配内容会高亮。
- **上一个 / 下一个**：跳转匹配位置。
- **区分大小写**：严格区分英文大小写。
- 在“替换为”中输入新内容，然后选择 **替换** 或 **全部替换**。
- 在查找输入框按 `Enter` 跳到下一个，按 `Shift + Enter` 跳到上一个。
- 点击面板右侧 `×` 关闭查找。

## 7. 全屏专注模式

点击全屏按钮后，标题栏、工具栏、标签栏、侧边栏和状态栏都会隐藏。Android 还会隐藏系统状态栏和导航栏。

退出方式：

- 点击右上角的退出全屏按钮。
- 按 Android 返回键。
- 外接键盘按 `Esc`。

## 8. 状态栏

底部状态栏依次显示：

- 当前行和列
- 文件语言类型
- 自动换行、左右滚动或大文件模式
- 行数和字符数
- 换行格式（LF、CRLF 或 CR）
- UTF-8 标识
- 已保存或未保存状态

软键盘弹出时，状态栏会自动隐藏以增加编辑空间。

## 9. 大文件模式

文件达到约 `750 KB` 后会自动进入大文件模式，以减少卡顿：

- 关闭语法高亮和自动换行。
- 跳过完整行数统计。
- 查找最多显示前 500 个匹配。
- Android 在后台读取文件。
- 最近记录只保存文件引用，下次打开时重新读取原文件。

因此不要移动或删除原文件，也不要撤销 Notepad+ 的文件访问权限。

## 10. 快捷键

`Ctrl` 在 macOS 上对应 `Command`。

### 应用快捷键

| 快捷键 | 功能 |
| --- | --- |
| `Ctrl + N` | 新建文本 |
| `Ctrl + O` | 打开文件 |
| `Ctrl + S` | 保存 |
| `Ctrl + Shift + S` | 另存为 |
| `Ctrl + F` | 打开查找和替换 |
| `Esc` | 关闭更多菜单、标签菜单或退出全屏 |

### 编辑快捷键

| 快捷键 | 功能 |
| --- | --- |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Y` | 重做；Linux 也可用 `Ctrl + Shift + Z`，macOS 使用 `Command + Shift + Z` |
| `Ctrl + A` | 全选 |
| `Tab` / `Shift + Tab` | 增加 / 减少缩进 |
| `Ctrl + /` | 切换行注释（支持的代码语言） |
| `Alt + ↑` / `Alt + ↓` | 上移 / 下移当前行 |
| `Shift + Alt + ↑` / `Shift + Alt + ↓` | 向上 / 向下复制当前行 |
| `Ctrl + Shift + K` | 删除当前行 |
| `Ctrl + Enter` | 插入空行 |

## 11. Android 返回键顺序

返回键会依次处理当前最上层界面：关闭标签菜单 → 关闭更多菜单 → 退出全屏 → 关闭查找 → 关闭侧边栏 → 返回首页 → 退出应用。

## 12. 常见问题

- **文字乱码**：文件可能不是 UTF-8 编码，建议先转换为 UTF-8。
- **无法覆盖保存**：原文件没有写入权限，请使用“另存为”。
- **最近文件打不开**：文件可能已移动或访问权限失效，请重新选择文件。
- **大文件功能减少**：这是自动性能保护，退出大文件模式需要打开更小的文件。
